Tkinterとは？

PythonでGUIアプリケーションを作成することのできるツールキットのことです。
）

GUI(グラフィカルユーザインタフェース)


コンピューターの操作の対象が絵で表現されるユーザーインターフェース。マウスなどのポインティング・デバイスを使用して直感的にコンピューターを操作できる。現在のOSやアプリケーションの多くは、GUIを採用している。


"Tkinter"を使用することによって、Pythonの実行ファイルをGUIとして実行することができます。

また、基本的にはPythonをインストールすると自動的に入っているツールなので、Pythonをインストールしていない方は、先にPythonのインストールから行ってください。

Pythonの環境構築から文法などの知識まですべてこの一冊で網羅することができていますので、APPNOTEでPython学習する際はこの参考書をオススメします！

Tkinterの使い方

まず、Tkinterを使用するにあたって、次のプログラミングコードを打ってください。 このコードは決まり文句のような意味合いもあるのでコピペしてもらっても大丈夫です。 


from tkinter import *
from tkinter import ttk

class calculator(ttk.Frame):

    def __init__(self, master=None):
        super().__init__(master)

def main():
    root = Tk()
    root.title('電卓アプリ')
    calculator(root)
    root.mainloop()


if __name__ == '__main__':
    main()

では、それぞれについて解説していきます。 



from tkinter import *
from tkinter import ttk

　１行目はTkinterのモジュールをインポートしている部分です。 「＊」を用いることですべてのモジュールをインポートできます。便利な使い方なので覚えておくといいかもしれませんね。
　２行目は、ttkというテーマが付いたウィジェットを読み込む部分です。これを使うことによって様々なウィジェットが使用できるようになります。



class culculator(ttk.Frame):

    def __init__(self, master=None):
        super().__init__(master)

この部分にボタンなどを配置する処理を書いていきますが、現段階ではなにも書いていません。
またttk.frameにボタンなどを配置していくことになります。 



def main():
    root = Tk()
    root.title('電卓アプリ')
    culculator(root)

ここでは作成したアプリが表示される画面(ウィジェット)を作成しています。

5行目は3行目で作成したウィジェットの中にボタンなどを配置する枠組みを作成しています。(基本的にこの上にボタンなどを直接配置していきます）
6行目はウィンドウ作成を実行する部分です。

電卓アプリのレイアウトを構成する

では、さっそく電卓の１～９のボタンを作成していきましょう。



class calculator(ttk.Frame):

  　def __init__(self, master=None):
        super().__init__(master)
        self.create_widgets()

　  def create_widgets(self):

        # 1列目
  　    ttk.Button(self, text='1').grid(column=0, row=0)
        ttk.Button(self, text='2').grid(column=1, row=0)
        ttk.Button(self, text='3').grid(column=2, row=0)

        # 2列目
        ttk.Button(self, text='4').grid(column=0, row=1)
        ttk.Button(self, text='5').grid(column=1, row=1)
        ttk.Button(self, text='6').grid(column=2, row=1)

        # 3列目
    　  ttk.Button(self, text='7').grid(column=0, row=2)
    　  ttk.Button(self, text='8').grid(column=1, row=2)
        ttk.Button(self, text='9').grid(column=2, row=2)

        # calculator自身もウィジェットに配置
        self.grid(column=0, row=0)

8行目でcreate_widgets()を定義し10~26行目でその内容を記述していきます。
そして6行目で内容をウィジェットに反映させます。

ttk.Button(self, text='〇').grid(column=〇, row=〇)は、ウィジェットにボタンを生成しています。
text=''の部分にボタンの名前、column=の部分に列番号、row=の部分に行番号を記述します。

また、grid()はラベルやボタンなどを、ウィンドウ上に配置していくのに使う関数の１つです。
grid()はグリッド(格子)状に配置することができるという特徴を持っています。

26行目はcalculator自身もウィジェットに配置しなければならないので、それが書かれています。
この記述を忘れると表示されないので気をつけてください。
